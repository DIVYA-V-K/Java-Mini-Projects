# Java Mini Projects

This repository contains a collection of core Java mini projects developed to strengthen fundamental and advanced programming concepts.

These projects demonstrate Object-Oriented Programming, Collections Framework, Stream API, Lambda Expressions, File Handling, Exception Handling, and Multithreading.

---

## 📚 Projects Included

### 1️⃣ Library Item Management System
- Demonstrates inheritance and polymorphism
- Uses package structuring
- Implements collection-based storage
- Shows access modifier usage

### 2️⃣ Student Course Registration System
- Uses Collections Framework
- Stream API filtering and grouping
- Lambda expressions for sorting
- Custom exception handling
- Menu-driven console application

### 3️⃣ Multi-Threaded Log File Analyzer
- Reads log file using File Handling
- Uses multiple threads for processing
- Counts INFO, ERROR, WARNING logs
- Combines results using thread coordination

---

## 🛠 Technologies Used
- Java (Core)
- OOP Concepts
- Collections Framework
- Stream API
- Lambda Expressions
- Multithreading
- File I/O

---

## 🚀 How to Run

1. Clone the repository
2. Open project in IntelliJ IDEA
3. Run individual main classes:

- `LibraryApp` → Library System
- `StudentMain` → Student Registration
- `LogMain` → Log Analyzer

Ensure `log.txt` is present in project root for Log Analyzer.
